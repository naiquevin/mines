Minesweeper

    @laptop/touchpad users: if pressing both buttons together causes scrolling, press one button first and then the other keeping the first one pressed.

    This is a javascript version of the famous windows game Minesweeper or Mines. It has been coded using Jquery. Thus the image of jquery logo for mines!

    Current version may still be buggy. Please report bugs / send suggestions to naikvin [at] gmail [dot] com.
